📆 第1周：代谢排毒阶段（第1-7天） 目标：温经散寒、利尿消肿、养血补铁

早餐：红糖小米粥 / 生化汤（100ml） / 藕粉羹 / 紫菜蛋丝馄饨

午餐：排骨汤细面 / 黑木耳豆腐肉丝蔬菜鸡蛋面 / 米饭 + 各类炒菜

晚餐：香菇瘦肉粥 / 蔬菜瘦肉粥 / 四神猪肝粥（剖腹产）

加餐：红糖小米粥 / 200g水果 / 红豆浓汤 / 核桃粉羹

煲汤：菠菜猪肝汤、清炖瘦肉汁、红萝卜蜜枣瘦肉汤

注意：剖腹产未排气前不宜进食，6小时后可从萝卜汤开始

📆 第2周：补肾壮腰、健脾阶段（第8-14天） 目标：收缩子宫、调养气血、促进排乳

早餐：红糖二米粥 / 牛奶燕麦粥 / 四神粥 / 牛肉粥

午餐：米饭 / 芝麻米饭 / 黑米饭 + 腰花、牛肉、炒菜

晚餐：小米淮山饭 / 黑芝麻米饭 + 各类荤素菜肴

加餐：纯牛奶 + 水果 / 坚果 / 红豆莲子茯苓汤

煲汤：木耳猪腰汤、杜仲猪腰汤、清炖瘦肉/鸡汁、益母草黑木耳汤

注意：2周后方可吃酒糟、鸽子汤

📆 第3-6周：调理巩固阶段（第15-42天） 目标：全面营养、恢复体力、母乳质量提升

特点：食材种类更加丰富，逐步恢复正常饮食

主食：五谷饭、薏仁饭、各类杂粮饭

蛋白质：增加鱼类（除生鱼）、禽类、豆制品

蔬菜：多样化时令蔬菜

汤品：继续配合药材汤品调理

注意事项：继续保持温补原则，避免生冷油腻

重要提醒与禁忌

⚠️ 严格禁忌

  • 生鱼：避免使用，可能导致伤口长肉芽
  • 凉食：所有食物应温热食用，避免寒凉
  • 油脂过多：易导致堵奶，应清淡不油腻

📋 剖腹产特别注意事项

  • 未排气前不宜进食
  • 6小时后可吃少许萝卜汤帮助排气
  • 排气后由流食、半流食，逐渐过渡到正常月子餐
  • 5-7天后方可吃红糖
  • 2周后方可吃酒糟、鸽子汤
  • 四神猪肝粥（薏仁、通心莲、茨实、山药）适合未排气后食用

💊 生化汤服用说明

  • 一般第二天开始服用（或遵医嘱）
  • 剖腹产喝2周，顺产喝1周
  • 每天300ml（分三次喝）
  • 若医生已开产妇康、益母草等药物，可不服用生化汤

🌡️ 食材温性参考

  • 温性水果：苹果、柚子、香蕉、木瓜、火龙果、葡萄、樱桃
  • 温性食材：小米、羊肉、鸡肉、核桃、花生、芝麻
  • 凉性食材（慎用）：绿豆、苦瓜、冬瓜、西瓜、梨

💡 实用建议

  • 食材尽量新鲜，少用冷冻食品
  • 烹饪以蒸、煮、炖为主，避免油炸
  • 每餐保持营养均衡：主食 + 蛋白质 + 蔬菜
  • 少量多餐，一天6餐为宜
  • 多喝水，可配合月子茶饮
  • 根据个人体质和医生建议调整食谱
